Welcome! To win a game of MEMORY you must pair two like cards before running out of turns.

Click any card to flip it. Try to remember what each card type is - it will increase you chances of pairing two cards of the same type.

To begin, enter the number of rows & columns you would like to play with. If you do not enter a number, the grid will default to 4 X 4.
